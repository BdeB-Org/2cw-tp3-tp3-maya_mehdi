Informations additionnelles sur notre projet

Ce site web a été créé pour la réalisation d'un travail pratique dans le cadre du cours "Applications web". Ce projet vise à se familiariser avec les notions vues dans le cadre du cours, c'est-à-dire Git, SQL et Javascript. Nous avons décidé de créer un site web pour les amateurs de technologie et plus précisément ceux pour qui les ordinateurs performants sont une passion. Dans ce site, on offre l'option d'acheter des composantes afin d'améliorer un ordinateur existant ou de se lancer dans la construction d'un nouvel ordinateur, mais on vend aussi des ordinateurs préfabriqués pour les personnes moins manuelles qui veulent un ordinateur performant sans avoir à le construire soi-même.
